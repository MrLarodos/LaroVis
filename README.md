# LaroVis

LaroVis: Ein leichtes und sauberes Web-Dashboard für ioBroker.

LaroVis bietet eine schnelle, ressourcenschonende und übersichtliche Visualisierung für Dein Smart Home. Es ist so aufgebaut, dass es leicht verständlich und ohne großen Aufwand an die eigenen Bedürfnisse anpassbar ist.

## Features

*   **Leichtgewichtig:** Reines HTML, CSS und JavaScript ohne überladene Frameworks.
*   **Modularer Aufbau:** Klare Trennung in einzelne Dateien für eine fehlerfreie und effiziente Weiterentwicklung.
*   **Multi-Language:** Eingebaute Unterstützung für mehrere Sprachen - aktuell Deutsch und Englisch.
*   **Smarte UI-Elemente:** Enthält praktische und moderne Kacheln wie "Statusregler" für eine intuitive Bedienung.
*   **Einfache Anbindung:** Kommuniziert direkt und unkompliziert mit den Datenpunkten Deines ioBroker-Systems.

## Installation / Nutzung

1. Lade die Dateien herunter oder klone das Repository.
2. Öffne die `index.html` im Browser.
3. Trage die IP-Adresse und den Port Deines ioBrokers ein.
4. Lege über das Zahnrad unten rechts Deine spezifischen Datenpunkte an (z.B. für PV-Ertrag, Batteriespeicher, Heizstab oder Wallbox) - fertig!
